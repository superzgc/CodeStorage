# CodeStorage
Excel Application by VSTO

This is a simple excel extension tool to store code snippet of coding practice.
Feature:

Sheet area: all data, each row for 1 problem.
Right panel: Enlarged text box for easy viewing.
For each problem, multiple version of code can be stored, and loaded out to file later.
Different tag can be assigned to each problems.
Content can be changed and selected by condition easily in sheet.
New tag and language can be added to corresponding sheets
